== Description ==

This is an addon for the Booking & Appointment Plugin for WooCommerce which lets you to select multiple time slots on a date for each product on the website.

== Installation ==

1. Download the zip file of the plugin.
2. Go to WordPress Admin > Plugins > Add New to upload.
3. Click Upload Plugin at the top.
4. Click Choose File and select the .zip file you downloaded in step 1. Select Choose.
5. Click Install Now.
6. Activate the plugin.
